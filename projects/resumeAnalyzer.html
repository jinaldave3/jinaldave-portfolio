    • Resume Analyzer Chrome Extension

    The Resume Analyzer is a Chrome extension designed to help job seekers instantly evaluate how well their resume aligns with any job description directly on platforms like LinkedIn or Indeed. Built using HTML, CSS, and JavaScript with Chrome Manifest V3, the extension allows users to upload their resume once, stores it locally using LocalStorage, and dynamically extracts job text from active pages via the chrome.scripting API. It then provides a real-time similarity score and match feedback within a clean, responsive popup interfaces, eliminating the need for external tools or manual comparisons.
