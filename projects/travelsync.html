    • TravelSync: A Smart Travel, Event & Networking Planner

    TravelSync is a full-stack travel planning platform designed to help users create trips, discover events, explore places, manage networking contacts, and generate AI-powered travel plans.

    The platform solves the problem of fragmented travel planning by combining trip organization, event discovery, destination research, networking follow-ups, and AI itinerary generation into one centralized dashboard.

    🎯 Objectives

    • 🧭 Create and manage trips with dates, purpose, budget, and notes.
    • 🎫 Discover local events near a destination using live event data.
    • 📍 Explore restaurants, attractions, museums, hotels, parks, and cafes.
    • 🤝 Track networking contacts and follow-up status after events.
    • 🤖 Generate AI-powered travel plans, packing lists, and event strategies.
    • 📌 Save places, events, and AI-generated plans into existing trips.

    🛠️ Technologies Used

    🖥️ Frontend Development

    • ⚛️ React.js – Component-based frontend architecture.
    • ⚡ Vite – Fast frontend development and build tooling.
    • 📡 Axios – Handles frontend-to-backend API requests.
    • 🎨 CSS3 – Custom responsive SaaS-style interface.
    • 🧩 State Management – Manages trips, contacts, places, events, and AI output.

    🌍 Backend & APIs

    • 🟢 Node.js – Backend runtime environment.
    • 🚂 Express.js – REST API backend server.
    • 🤖 OpenAI API – AI itinerary generation and travel recommendations.
    • 🎫 Ticketmaster Discovery API – Live event discovery by city and keyword.
    • 📍 Geoapify Places API – Destination search for attractions, restaurants, hotels, parks, and more.

    📊 Data Engineering & DevOps

    • 🐘 PostgreSQL Schema – Structured data model for trips, itinerary items, contacts, and saved places.
    • 🐍 Python ETL Starter – Data transformation starter for travel and event analytics.
    • 🌬️ Apache Airflow – Pipeline orchestration structure.
    • 🐳 Docker – Containerized application setup.
    • 🔄 GitHub Actions – CI workflow for frontend and backend checks.

    📂 Project Structure

    🖥️ Frontend Pages

    • 🏠 Home Dashboard – Displays trip stats, itinerary count, contacts, and follow-ups.
    • 🧭 Trips – Create trips, edit trip details, add notes, and view saved itinerary items.
    • 🎫 Events – Search live events and add selected events to existing trips.
    • 📍 Places – Search places by destination and category, view details, and add places to trips.
    • 🤝 Network – Save networking contacts and update follow-up status.
    • 🤖 AI Assistant – Generate travel plans and save them into existing trips.

    🖧 Backend Services

    • 🧭 trips.js – Handles trip creation, editing, and itinerary saving.
    • 🎫 eventService.js – Fetches live event data from Ticketmaster.
    • 📍 placesService.js – Fetches destination places from Geoapify.
    • 🤖 aiService.js – Sends travel planning prompts to OpenAI.
    • 🤝 contacts.js – Manages networking contacts and follow-up statuses.
    • 📊 analyticsService.js – Calculates dashboard metrics.

    🚀 Features & Functionality

    • 🧭 Trip Creation & Editing – Users can create trips, update details, and add notes.
    • 📌 Saved Itinerary Items – Places, events, and AI plans can be saved into trips.
    • 🎫 Live Event Discovery – Users can search events by destination and keyword.
    • 📍 Places Explorer – Users can find attractions, restaurants, cafes, museums, hotels, airports, and parks.
    • 🤝 Networking CRM – Users can save contacts and track follow-up progress.
    • 🤖 AI Travel Assistant – Generates itinerary suggestions, packing lists, and travel strategies.
    • 🛡️ Graceful API Handling – Provides clear user-facing messages when APIs are unavailable or quota-limited.
