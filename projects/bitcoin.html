    • Bitcoin Price Comparison Tool: A React-Based Web Application

    Cryptocurrency trading requires real-time price tracking to ensure users get the best rates when buying Bitcoin (BTC). The Bitcoin Price Comparison Tool is a React-based web application that enables users to compare BTC prices from multiple providers dynamically.

    This tool fetches real-time BTC prices, sorts the results, and displays them in an interactive and user-friendly interface. The application leverages TypeScript, React.js, Axios, and Node.js to ensure fast performance and accurate pricing data.

    🎯 Objectives

    • 🚀 Provide an efficient and user-friendly BTC price comparison tool.
    • 💰 Fetch and display real-time Bitcoin conversion rates from multiple providers.
    • 🔢 Allow users to input an amount and receive dynamically updated rates.
    • ⚡ Ensure high performance, reliability, and smooth UI interactions.

    🛠️Technologies Used

    🖥️ Frontend Development

    • ⚛️ React.js + TypeScript – Component-based UI structure.
    • 🎨 Tailwind CSS – Provides modern styling and responsive design.
    • 🔀 Lodash – Used for sorting price data efficiently.
    • 🔄 Debounced Effect – Ensures smooth API calls without excessive requests.

    🌍 Backend & Data Fetching

    • 🌐 Aircode (Node.js Serverless Functions) – Handles backend logic for fetching BTC rates.
    • 📡 Axios – Fetches real-time data from external APIs.
    • ⚙️ Node.js (v18) – Provides runtime support for serverless functions.

    🛠️ Development Tools & Configuration

    • 📦 TypeScript – Ensures code maintainability and type safety.
    • 🔧 ES2022 (via TypeScript Config) – Uses modern JavaScript features.
    • ⚡ Vite – Optimized development and build tool for React.

    📂 Project Structure

    🖥️ Frontend Components

    • 🏠 App.tsx – Main UI logic, fetches BTC offers, and manages state.
    • 💲 AmountInput.tsx – User input field for entering the amount.
    • 🔄 LoadingSkeleton.tsx – Displays a loading animation while fetching data.
    • 📊 ResultRow.tsx – Displays BTC offers from different providers.

    🖧 Backend API Handling

    • 🌍 offers.ts – Fetches real-time BTC offers from external providers.
    • 🛢️ cachedValues.ts – Stores previously fetched BTC rates for efficiency.
    • 🌍 providers.ts – Defines supported BTC exchange providers.
    • 🛠️ cache.ts – Implements caching logic for API responses.

    ⚙️ Configuration Files

    • 📝 aircode.config.json – Configures serverless runtime (Node.js v18).
    • 📦 package.json – Lists dependencies like Axios, Aircode, TypeScript.
    • 🔧 tsconfig.json – Defines TypeScript compiler settings for ES2022.

    🚀 Features & Functionality

    • 🔍 Real-Time BTC Price Comparison – Fetches and sorts BTC rates dynamically.
    • 📥 User Input for Custom Amounts – Adjust BTC conversion amount easily.
    • 🎛 Interactive UI – Tailwind CSS ensures a smooth and modern design.
    • 🔄 Optimized API Requests – Uses debouncing to prevent excessive API calls.
    • ⚡ Serverless Backend (Aircode) – Ensures efficient data retrieval.
    • 📡 Cached Values for Performance – Stores previous API responses to speed up loading.
