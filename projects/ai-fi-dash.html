    • AI Financial Insights Dashboard: A Full-Stack Fintech Web Application

    The AI Financial Insights Dashboard is a modern fintech-style web application designed to provide users with real-time stock market data, AI-generated insights, and interactive financial visualizations.

    The platform combines live financial APIs with AI-powered analysis to help users quickly understand stock performance, market sentiment, and trending financial news through a clean and responsive dashboard experience.

    🎯 Objectives

    • 📈 Provide a modern dashboard for viewing real-time stock market information.
    • 📰 Deliver live company-specific financial news and market updates.
    • 🤖 Generate AI-powered market insights and stock summaries.
    • 📊 Visualize historical stock performance using interactive charts.
    • 🎨 Create a polished fintech-inspired user experience.

    🛠️ Technologies Used

    🖥️ Frontend Development

    • ⚛️ React.js – Component-based frontend architecture.
    • 🎨 Tailwind CSS – Responsive fintech-style UI design.
    • 📡 Axios – Handles frontend API requests.
    • 📈 Recharts – Interactive stock price chart visualizations.
    • ⚡ Vite – Fast frontend build and development tooling.

    🌍 Backend & APIs

    • 🟢 Node.js – Backend runtime environment.
    • 🚂 Express.js – REST API backend server.
    • 💹 Finnhub API – Real-time stock quotes, company profiles, and financial news.
    • 📊 Yahoo Finance API – Historical stock chart data.
    • 🤖 OpenAI API – AI-generated stock insights and summaries.

    📂 Project Structure

    🖥️ Frontend Components

    • 🏠 App.jsx – Main dashboard UI and state management.
    • 🔍 SearchBar.jsx – Stock ticker search functionality.
    • 📈 PriceChart.jsx – Interactive stock chart rendering.
    • 📰 NewsList.jsx – Displays real-time financial news.
    • 📊 KPIGrid.jsx – Market metrics and stock KPIs.
    • 🤖 InsightCard.jsx – AI-generated market insight display.

    🖧 Backend Services

    • 🌍 stocks.js – Backend routes for stock dashboard data.
    • 📡 stockService.js – Handles financial API integrations.
    • 🤖 aiService.js – Generates AI-powered market analysis.
    • ⚙️ index.js – Express server configuration.

    🚀 Features & Functionality

    • 🔍 Real-Time Stock Search – Search and analyze US stock tickers dynamically.
    • 📈 Interactive Market Charts – Visualize historical stock performance.
    • 📰 Live Financial News Feed – Fetch latest company-specific market news.
    • 🤖 AI-Powered Insights – Generate stock summaries, risks, and opportunities.
    • 🎨 Modern Fintech UI – Responsive dashboard with polished user experience.
    • ⚡ API Integration Architecture – Combines multiple financial APIs into a unified dashboard.
