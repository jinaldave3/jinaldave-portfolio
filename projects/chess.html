    • Java Chess Game: A Desktop-Based Chess Application

    The Java Chess Game is a desktop-based chess application developed using Java Swing. It provides an interactive and visually appealing chess-playing experience, supporting player vs. player mode, move validation, an undo feature, and standard chess rules. The game includes an object-oriented architecture where each chess piece is an independent class with its own movement logic.
