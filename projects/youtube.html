    Project Structure

    1. Header (header.css)

    The header includes: - YouTube logo and a hamburger menu. - A search bar with button functionality and voice search. - Icons for notifications, uploads, and user profile.

    2. Sidebar (sidebar.css)

    The sidebar enables seamless navigation with links to: - Home, Explore, Subscriptions, Originals, YouTube Music, and Library. - Uses hover effects for better user interactivity.

    3. Video Grid (video.css)

    The main content area displays a grid layout of video previews, each containing: - Thumbnail with video duration overlay. - Channel picture, video title, author, and video stats. - Uses CSS Grid to adjust layout based on screen size.


    Responsive Design

    The website adapts dynamically to different screen sizes using media queries: - Large screens (4-column layout). - Medium screens (3-column layout). - Small screens (2-column layout).
